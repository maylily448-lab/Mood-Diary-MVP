
# 🌤️ Mood Diary — 감성 무드 트래킹 앱

20대를 위한 미니멀하고 감성적인 감정 기록 웹앱을 만듭니다.

---

## 디자인 시스템

- **테마**: 소프트 파스텔 스카이블루 (sky-50 ~ sky-100 그라데이션 배경)
- **카드**: 큰 카드 기반 레이아웃, `rounded-2xl`, 부드러운 그림자
- **타이포그래피**: 깔끔한 sans-serif, 친근한 톤
- **애니메이션**: Framer Motion으로 페이지 전환, 버튼 인터랙션 (hover/tap scale-up)

---

## 페이지 구성

### 1. 홈 (무드 기록)
- 5개의 큰 이모지 버튼: 😆 최고 / 😊 좋아 / 😐 보통 / 😢 슬퍼 / 😠 화나
- 각 버튼 탭/호버 시 scale-up 애니메이션
- 선택 후 "한 줄 메모" 입력 필드 표시
- '저장' 버튼

### 2. 저장 성공 피드백
- 저장 시 아름다운 모달 또는 토스트 표시
- 랜덤 응원 메시지 (예: "오늘도 수고했어요!", "당신은 충분히 잘하고 있어요 💛")

### 3. 히스토리 페이지
- 최근 7일간의 기록을 세로 카드 리스트로 표시
- 각 카드: 이모지 + 날짜 + 메모
- 부드러운 fade-in 애니메이션

### 4. 통계 (Insights) 페이지
- 이번 달 감정별 빈도를 Bar Chart로 표시 (Recharts 사용)
- 깔끔하고 파스텔 톤의 차트 색상

### 5. 하단 내비게이션 바
- 고정 하단 바: 홈 / 히스토리 / 인사이트
- Lucide 아이콘 사용
- 현재 페이지 하이라이트

---

## 기술 사항
- React + Tailwind CSS + Framer Motion (신규 설치)
- Lucide-react 아이콘
- Recharts 바 차트
- 로컬 상태 + 목업 데이터 (백엔드 연동 없음)
- 모바일 최적화, 데스크톱 대응 반응형
